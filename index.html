Sobre mí

Me gusta combinar estructura técnica, creatividad visual y enfoque de producto para construir soluciones que se sientan útiles y memorables.

Soy estudiante y desarrollador enfocado en software, desarrollo web y experiencias digitales. Desde mi etapa escolar he fortalecido mis bases en tecnología, programación y lógica computacional, y hoy sigo expandiendo ese camino con proyectos propios que me retan a construir mejor.

2026 Proyecto destacado

NixoCore

Producto propio | Automatización, software web e IA

NixoCore es mi proyecto orientado a producto digital, pensado para explorar automatización, organización de procesos y soluciones útiles para negocios. La idea del proyecto es construir una base SaaS con visión práctica, enfoque real y potencial de crecimiento.

En este proyecto trabajo estructura de producto, lógica de negocio, interfaz web y experimentación con IA aplicada a flujos de atención, automatización y soporte de procesos.

2025 Videojuego

Dungeon Trade

Desarrollador | Game Designer | UI Designer

Dungeon Trade es un videojuego ambientado en un mundo de fantasía medieval, donde la magia es un recurso escaso y valioso. La historia sigue a Belthazar, un mago artesano que busca sobrevivir, comerciar y adentrarse en una mazmorra en busca de recursos mágicos.

Este proyecto me permitió poner en práctica habilidades en desarrollo de videojuegos, diseño de interfaz, diseño de experiencia y construcción narrativa, además de trabajo colaborativo dentro de una Game Jam.

Unity C# Game Design UI Design Narrativa

2026 Backend / API

Mythic Clash API

Proyecto backend | API REST y lógica de combate

Mythic Clash API es una API REST construida para gestionar personajes RPG y simular batallas estratégicas. Fue desarrollada como proyecto académico para practicar backend, operaciones CRUD, persistencia de datos y lógica de negocio.

La arquitectura del proyecto separa rutas, servicios, repositorio y lógica del sistema de batalla, lo que me permitió practicar una organización más limpia del código.
